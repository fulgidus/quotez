4. **Server Lifecycle**: init() now handles socket setup, no separate listen()
5. **Selector API**: next() returns ?usize (optional), not error union
6. **Performance Thresholds**: Account for test harness overhead vs production paths
7. **Docker Verification**: Separate code completion from infrastructure verification
