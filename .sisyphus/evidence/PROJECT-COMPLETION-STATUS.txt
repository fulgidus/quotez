Key Learnings:
- Zig 0.16 removed std.net entirely - use std.posix for all sockets
- Timer API: std.time.Instant.now() is error union (must use try)
- sockaddr.in requires @ptrCast for socket function compatibility
- Server lifecycle: init() handles socket setup, no separate listen()
- Selector.next() returns ?usize (optional), not error union
- Performance thresholds should account for test harness overhead
